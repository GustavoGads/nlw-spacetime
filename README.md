## 💻💪 Projeto </>

Esse é um projeto de desenvolvimento Web Responsivo de uma cápsula do tempo para exibir memórias em uma timeline.

## 🚀⚙ Tecnologias </> 
Esse projeto foi desenvolvido durante o NLW da Rocktseat com as seguintes tecnologias:
- HTML
- CSS
- Git e Github
